Introduction

Roody is ROOT application to display histograms and other contents from ROOT and HBOOK files and from the online connection to the MIDAS and ROME online data analyzers. It is meant for display of .root files and .hbook files. If running this application in conjunction with either the Midas analyzer or the ROME analyzer, it will allow to visualize online data retrieved through a socket port.
